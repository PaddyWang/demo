CSS 变量

CSS 以 -- 作为命名前缀 , 用 var() 去引用

一般通过:root { }伪类 声明变量

注意: 声明的变量需要在同一个作用域内才能生效

e.g:

:root {

--blue: #1890ff;

--lightBlue: #bae7ff;

--pink: #ffd6e7;

}

color: var(--blue);

color: var(--lightBlue);

color: var(--pink);

attr 属性

a[href]:empty::before {

content: attr(href);

}
